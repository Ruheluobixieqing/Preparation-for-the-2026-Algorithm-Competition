# JZ76.删除链表中重复的结点

第一种思路，使用 map 存每个数出现的次数，第二次遍历的时候执行删除操作

这种方式的优点在于无论链表是否有序都可以处理，但是需要额外空间

```cpp
class Solution {
public:
    ListNode* deleteDuplication(ListNode* pHead) {
        std::map<int, int> mp;
        ListNode *p = pHead, *lastNode = pHead;
        while(p)
        {
            mp[p->val] ++;
            p = p -> next;
        }
        p = pHead;
        while(p)
        {
            if(mp[p->val] > 1)
            {
                if(p == pHead) pHead = p -> next;
                else
                {
                    lastNode -> next = p -> next;
                    p = p -> next;
                }
            }
            else 
            {
                lastNode = p;
                p = p -> next;
            }
        }
        return pHead;
    }
};
```

第二种方式，考虑到表是升序的，所以可以往后找，判断是否需要删除

```cpp
class Solution {
public:
    ListNode* deleteDuplication(ListNode* pHead) {
        ListNode *p = pHead, *lastNode = pHead;
        while(p)
        {
            if(p -> next == nullptr) break;
            if(p -> next -> val == p -> val)
            {
                ListNode *nextNode = p -> next;
                while(nextNode != nullptr && nextNode -> val == p -> val) nextNode = nextNode -> next;
                if(p == pHead) pHead = nextNode;
                else
                {
                    lastNode -> next = nextNode;
                    p = nextNode;
                }
            }
            else
            {
                lastNode = p;
                p = p -> next;
            }
        }
        return pHead;
    }
};
```
