# JZ24.反转链表

新建两个额外的指针，一个用于存储当前遍历到的结点，一个用于存储上一个遍历到的结点，然后改一下指针方向就可以了

```cpp
class Solution {
public:
    ListNode* ReverseList(ListNode* head) {
        if( head == nullptr ) return head;
        ListNode* p = head;
        ListNode *nowNode;
        ListNode *preNode = nullptr;
        while(p)
        {
            // 记录当前结点
            nowNode = p;
            // 记录下一个结点
            p = p -> next;
            nowNode -> next = preNode;
            preNode = nowNode;
        }
        return nowNode;
    }
};
```
