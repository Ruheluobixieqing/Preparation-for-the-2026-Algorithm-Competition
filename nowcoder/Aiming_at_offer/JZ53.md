# JZ53.数字在升序数组中出现的次数

用 lower_bound 和 upper_bound 快速定位双端即可

```cpp
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param nums int整型vector 
     * @param k int整型 
     * @return int整型
     */
    int GetNumberOfK(vector<int>& nums, int k) {
        auto left = std::lower_bound(nums.begin(), nums.end(), k);
        auto right = std::upper_bound(nums.begin(), nums.end(), k);
        return right - left;
    }
};
```
