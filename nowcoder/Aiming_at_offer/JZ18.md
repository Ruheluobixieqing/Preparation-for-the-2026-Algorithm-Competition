# JZ18.删除链表中的结点

由于题目保证了链表中各个结点的值不相同，因此删除一个结点后可以直接 return

```cpp
class Solution {
public:
    /**
     * 代码中的类名、方法名、参数名已经指定，请勿修改，直接返回方法规定的值即可
     *
     * 
     * @param head ListNode类 
     * @param val int整型 
     * @return ListNode类
     */
    ListNode* deleteNode(ListNode* head, int val) {
        ListNode *p = head, *lastNode = head;
        if(head -> val == val) return head->next;
        p = p -> next;
        while(p){
            if(p->val == val) 
            {
                lastNode -> next = p -> next;
                return head;
            }
            else 
            {
                lastNode = p;
                p = p -> next;
            }
        }
        return head;
    }
};
```
