# 108.将有序数组转换为平衡二叉搜索树

我们定义一个函数 sortedArrayToBST(), 传入参数为一个已经排好序的 vector<int> v

对于（平衡）二叉搜索树，其中序遍历结果必然是升序的

那么我们可以从 v 中找到中间结点作为树的根结点，然后将 v 分为 left_v 和 right_v，对左右部分递归调用函数进行处理

最终就可以得到结果

```cpp
class Solution {
public:
    TreeNode* sortedArrayToBST(vector<int>& nums) {
        int n = nums.size();
        if(n == 0) return nullptr;
        int mid = n >> 1;
        TreeNode* root = new TreeNode(nums[mid]);
        std::vector<int> leftNums(nums.begin(), nums.begin() + mid);
        std::vector<int> rightNums(nums.begin() + mid + 1, nums.end());
        TreeNode* left = sortedArrayToBST(leftNums);
        TreeNode* right = sortedArrayToBST(rightNums);
        root -> left = left;
        root -> right = right;
        return root;
    }
};
```
