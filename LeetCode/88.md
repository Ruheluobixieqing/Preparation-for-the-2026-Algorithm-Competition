# 88.合并两个有序数组

开一个新的 vector 容器，两个指针扫一遍按序存进去最后和 nums1 交换一下就可以了

这种方法可以保证时间复杂度最优，为 O(m + n)
```cpp
class Solution {
public:
    void merge(vector<int>& nums1, int m, vector<int>& nums2, int n) {
        int i = 0, j = 0, p = 0;
        std::vector<int>arr(m+n);
        while(p < n + m)
        {
            if(i >= m) arr[p++] = nums2[j++];
            else if( j >= n) arr[p++] = nums1[i++];
            else{
                if(nums1[i] <= nums2[j]) arr[p++] = nums1[i++];
                else arr[p++] = nums2[j++];
            }
        }
        std::swap(arr, nums1);
    }
};
```
