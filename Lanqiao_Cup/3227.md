# 3227.找到最多的数

- 矩阵中至多有 1 个数可以满足出现次数超过总个数一半的条件
- 满足条件则可以直接确定最终结果

```cpp
#include <bits/stdc++.h>
std::map<int, int>cal;

int main()
{
  int n,m;
  std::cin>>n>>m;
  int threshold = n * m / 2;
  int ans = 0;
  for(int i = 1; i <= n; i++)
  {
    for(int j = 1; j <= m; j++)
    {
      int k;
      std::cin>>k;
      cal[k]++;
      if(cal[k] > threshold) ans = k;
    }
  }
  std::cout<<ans<<'\n';
  return 0;
}
```
